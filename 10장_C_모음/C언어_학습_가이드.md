### [317 기출체크1.c] — switch fall-through

```c
int c = 1;
switch(3) {
    case 3: c = 0;    // c=0 (break 없음)
    case 4: c += 3;   // c=3 (break 없음)
    case 5: c -= 10;  // c=-7 (break 없음)
    default: c--;     // c=-8
}
```
> **출력:** `-8`  
> **포인트:** case 3부터 시작해서 break 없으므로 아래 모두 실행

### [320 포인터 기출체크1.c] — 포인터 연산

```c
*(ary + 0) = 1;        // ary[0] = 1
ary[1] = *(ary+0) + 2; // ary[1] = 3
ary[2] = *ary + 3;     // *ary = ary[0] = 1 → ary[2] = 4
s = 1+3+4 = 8
```
> **출력:** `8`

### [Section 120_기출 따라잡기_문제1.c] — 문자열 포인터

```c
char *p = "KOREA";
printf("%s\n", p);      // KOREA   (p[0]부터 \0까지)
printf("%s\n", p+3);    // EA      (p[3]부터 \0까지)
printf("%c\n", *p);     // K       (p[0] 문자)
printf("%c\n", *(p+3)); // E       (p[3] 문자)
printf("%c\n", *p+2);   // M       (K의 ASCII 75 + 2 = 77 = 'M')
```

### [문제41.c] — 값 전달 vs 참조 전달

```c
int a=11, b=19;
swap(a, b);          // 값 전달 → a,b 원본 불변 (a=11, b=19)
switch(a) {          // a=11
    case 11: b+=2;   // b=21 (break 없음)
    default: b+=3;   // b=24
    break;
}
printf("%d", a-b);   // 11-24 = -13
```
> **출력:** `-13`  
> **포인트:** 함수에서 `swap(int a, int b)`는 값 복사 → 원본 불변

## 핵심 암기 사항

```
진법         : 0으로 시작 → 8진수, 0x → 16진수
포인터       : *p = 역참조(값), &a = 주소
문자열포인터 : p+n → n번째 위치부터 출력
구조체포인터 : -> 연산자 (= (*p).멤버)
switch       : break 없으면 아래 case 모두 실행 (fall-through)
static 변수  : 함수 내에서도 호출 간 값 유지
값 전달      : 기본 자료형은 값 복사 → 원본 불변
참조 전달    : 포인터로 전달 시 원본 변경 가능
재귀         : 종료 조건 반드시 확인
동적 할당    : malloc → free 짝 맞추기
```
