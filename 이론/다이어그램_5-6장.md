## 5장: AJAX 동작 흐름 ★B

```mermaid
sequenceDiagram
    participant U as 사용자
    participant B as 브라우저
    participant AE as AJAX 엔진
    participant S as 서버

    U->>B: 이벤트 발생 (클릭 등)
    B->>AE: XMLHttpRequest 객체 생성
    AE->>S: 비동기 HTTP 요청
    Note over B: 페이지 리프레시 없음
    Note over B: 다른 작업 계속 가능
    S-->>AE: JSON/XML 데이터 응답
    AE->>B: DOM 부분 업데이트
    B->>U: 변경 부분만 화면 갱신
```

> **AJAX** = Asynchronous JavaScript And XML
> 비동기 방식으로 서버와 데이터 교환, 전체 페이지 리로드 없이 부분 업데이트
