## 파일별 상세 풀이

### [문제15.py] — for + range 누적합

```python
hap = 0
for i in range(1, 11):   # i: 1~10
    hap += i
print(i, hap)             # i는 루프 종료 후 10 유지
```

> **출력:** `10 55`  
> **포인트:** 루프 변수 `i`는 루프 종료 후에도 마지막 값(10) 유지

### [문제12.py] — continue로 홀수 건너뛰기

```python
a = sum = 0
while a < 10:
    a += 1
    if a % 2 == 1:
        continue    # 홀수면 skip → sum에 더하지 않음
    sum += a        # 짝수만 합산: 2+4+6+8+10
print(sum)
```

> **출력:** `30`

### [문제 26.py / 문제27.py] — while + break 1부터 100까지 합

i, hap = 0, 0
while True:
    i += 1
    hap += i
    if i >= 100:

### [기출 따라잡기_문제1.py] — 리스트 뒤집기 + 슬라이싱

```python
def func(lst):
    for i in range(len(lst) // 2):   # 0, 1, 2
        lst[i], lst[-i-1] = lst[-i-1], lst[i]   # swap

lst = [1, 2, 3, 4, 5, 6]
func(lst)
# 뒤집기 후: [6, 5, 4, 3, 2, 1]
print(sum(lst[::2]) - sum(lst[1::2]))
# lst[::2]  = [6, 4, 2]  → 합 12
# lst[1::2] = [5, 3, 1]  → 합 9
# 12 - 9 = 3
```

> **출력:** `3`

### [기출 따라잡기_문제2.py] — type() 비교

```python
def func(value):
    if type(value) == type(100):     # int인지 확인
        return 100
    elif type(value) == type(""):    # str인지 확인
        return len(value)
    else:
        return 20

a = "100.0"   # str → len("100.0") = 5
b = 100.0     # float → else → 20
c = (100, 200)# tuple → else → 20
print(func(a) + func(b) + func(c))  # 5 + 20 + 20
```

> **출력:** `45`

### [330 Python의 클래스.py] — 클래스 메소드

```python
class Cls:
    x, y = 10, 20
    def chg(self):
        temp = self.x
        self.x = self.y
        self.y = temp

a = Cls()
print(a.x, a.y)   # 10 20
a.chg()
print(a.x, a.y)   # 20 10
```

### [335 기출체크1.py] — 딕셔너리 컴프리헨션 + 세트 교집합

```python
lst = [1, 2, 3]
dst = {i: i * 2 for i in lst}   # {1:2, 2:4, 3:6}
s = set(dst.values())            # {2, 4, 6}
lst[0] = 99                      # lst 변경 (dst엔 영향 없음)
dst[2] = 7                       # dst[2]: 4 → 7
s.add(99)                        # s: {2, 4, 6, 99}

# dst.values() = {2, 7, 6}  (set으로 변환)
# s & {2, 7, 6} = {2, 6} (교집합)
print(len(s & set(dst.values()))) # 2
```

> **출력:** `2`

### [문제46.py] — 문자열 패턴 카운트

```python
def cnt(str, p):
    result = 0
    for i in range(len(str)):
        sub = str[i:i+len(p)]   # 슬라이딩 윈도우
        if sub == p:
            result += 1
    return result

str = "abdcabcabca"
p1 = "ca"    # "abdcabcabca"에서 ca 찾기 → 2번
p2 = "ab"    # "abdcabcabca"에서 ab 찾기 → 3번
print(f'ab{cnt(str, p1)} ca{cnt(str, p2)}')
```

> **출력:** `ab2 ca3`

## 핵심 암기 사항

```
슬라이싱      : 문자열[시작:끝]  ← 끝 인덱스 미포함
음수 인덱스   : 문자열[-1] = 마지막 문자
세트          : 중복 없음, 순서 없음
range(1, 11)  : 1~10 (끝 미포함)
print end     : end='' 는 줄바꿈 없애기
lambda        : lambda 변수: 표현식
type 비교     : type(x) == type(100) → 정수 확인
```
